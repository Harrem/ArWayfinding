AR Indoor Navigation App

About the App

The AR Indoor Navigation App is an innovative solution designed to revolutionize navigation within Charmo University campus. Leveraging augmented reality (AR) technology, the app provides users with a seamless and immersive indoor navigation experience.

Enhanced Features

  • Real-time navigation guidance
  • Interactive 3D representation of buildings
  • Accurate location tracking
  • Search functionality for specific rooms or destinations
  • Multiple language support
  • Customizable user preferences
